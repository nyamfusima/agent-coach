# Essentials — Apply Credit

**Systems used:** Carehub, Hansen

1. [Carehub] Very if customer is eligible
2. [Hansen] Select Apply Financial Transaction
3. [Hansen] Select Apply credit on drop down
4. [Carehub] Get Authorization from Manager
5. [Hansen] Complete the required fields
6. [Hansen] Click next
7. [Hansen] Fill in unbilled credit request
8. [Hansen] Select the product
9. [Hansen] Enter Reference
10. [Hansen] Enter refund amount
11. [Hansen] Click next
   - (end of this flow)
12. [Hansen] Click next x 3
13. [Hansen] Click return to customer
   - (end of this flow)
14. **Decision — [Carehub] Load Bundle?**
   - If **NO**: No Action
15. [Carehub] No Action
   - (end of this flow)
