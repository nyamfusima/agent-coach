# Device Support — Troubleshooter

**Systems used:** Device Help

1. [Device Help] Identify the issue details
2. [Device Help] Follow the prompts
3. **Decision — [Device Help] Resolved?**
   - If **NO**: Under Warranty
4. **Decision — [Device Help] Under Warranty?**
   - If **YES**: Book for repair
   - If **NO**: Inform Customer
5. [Device Help] Inform Customer
6. **Decision — [Device Help] Proceed?**
   - If **YES**: Book for repair
7. [Device Help] Book for repair
8. [Device Help] Copy ref number to add to notes and share with customer
   - (end of this flow)
